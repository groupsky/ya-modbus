# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.5.x   | :white_check_mark: |
| < 0.5   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by emailing 185580+groupsky@users.noreply.github.com.

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

You can expect an initial response within 72 hours. We will keep you informed of progress toward a fix and may ask for additional information.

Please do not disclose the vulnerability publicly until we have had a chance to address it.
